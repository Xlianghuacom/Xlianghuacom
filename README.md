## 📈 核心策略类型
### 🔸 稳健型策略（低波动·稳收益）
- **资金费率套利**：本所套利（现货+永续合约组合）、跨所费率差套利（无净持仓风险）  
- **价差套利**：跨所价差（同资产跨平台低买高卖）、期现套利（基差回归）、期权套利（定价偏差组合）  
- **高频做市商**：提供流动性赚取价差，高频调整订单严控库存风险  


### 🔸 波动性策略（抓趋势·搏波动）
- **CTA策略**：多空双向，量化模型捕捉全品种趋势  
- **趋势策略**：覆盖短/中/长期趋势跟踪  
- **统计套利**：基于资产历史相关性，偏离均值建仓（配对交易/跨品种套利）
